The Code for FIZZ BUZZ. 

                  
                    
                    // fizz buzz
                    // write a short program that displays each number from 1-100
                    // for each multiple of 3, display "Fizz" instead of the number
                    // For each multiple of 5, display "Buzz" instead of the number
                    // For numbers which are multiples of both 3 and 5 display "FizzBuzz" instead of the number.
                    
                    function getValues() {
                      //get the user input for the fizz number
                      let fizzValue = document.getElementById("fizzValue").value;
                      console.log(fizzValue);
                    
                      //get the user input for the buzz number
                      let buzzValue = document.getElementById("buzzValue").value;
                    
                      //Convert "fizzValue" to an integer using parseInt()
                      fizzValue = parseInt(fizzValue);
                    
                      //Convert "buzzValue" to an integer using parseInt()
                      buzzValue = parseInt(buzzValue);
                    
                      //validate that the data are integers
                      if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {
                        //pass fizzValue and buzzValue to the FizzBuzz function and strore result in variable
                        let fbData = FizzBuzz(fizzValue, buzzValue);
                    
                        //pass fbData to the displayData function, display the fizzbuzz results to the page
                        displayData(fbData);
                      } else {
                        // alert the user only integers must be entered
                        alert("You must enter integers");
                      }
                    }
                    
                    function FizzBuzz(value1, value2) {
                      //create empty array to store the fizzbuzz data
                      let returnArray = [];
                    
                      // iterate through numbers 1-100, checking for fizz buzz values, push numbers and fizzbuzz values into the returnArray
                      for (let i = 0; i <= 100; i++) {
                        // if i divisible by 3 and 5
                        if (i % value1 == 0 && i % value2 == 0) {
                          // fizzbuzz
                          returnArray.push("FizzBuzz");
                        }
                        // if i is divisible by only 3
                        else if (i % value1 == 0) {
                          //fizz
                          returnArray.push("Fizz");
                        }
                        // if i is divisible by only 5
                        else if (i % value2 == 0) {
                          //Buzz
                          returnArray.push("Buzz");
                          // not divisible by 3 or 5
                        } else {
                          // push the  index
                          returnArray.push(i);
                        }
                      }
                    
               
                      return returnArray;
                    }
                    
                    //custom display function
                    function displayData(fbData) {
                      //get the table body element from the page
                      let tableBody = document.getElementById("results");
                      //get the template row
                      let templateRow = document.getElementById("fbTemplate");
                      //clear table first
                      tableBody.innerHTML = "";
                    
                      //loop over the array and create a table row for each item.
                      for (let i = 1; i <= fbData.length; i += 5) {
                        //add the className to the element
                        //imports template as a document fragment to make a copy of the template in memory so it can be modified and added into the table tag in the HTML, content is the tr tag nested inside the template
                        let tableRow = document.importNode(templateRow.content, true);
                        // grab the tds and put them into the array
                        let rowCols = tableRow.querySelectorAll("td");
                        // add the respective class to each column to change the styling
                        // set the data for each column
                        rowCols[0].classList.add(fbData[i]);
                        rowCols[0].textContent = fbData[i];
                        //
                        rowCols[1].classList.add(fbData[i + 1]);
                        rowCols[1].textContent = fbData[i + 1];
                        //
                        rowCols[2].classList.add(fbData[i + 2]);
                        rowCols[2].textContent = fbData[i + 2];
                        //
                        rowCols[3].classList.add(fbData[i + 3]);
                        rowCols[3].textContent = fbData[i + 3];
                        //
                        rowCols[4].classList.add(fbData[i + 4]);
                        rowCols[4].textContent = fbData[i + 4];
                        // add the template to the document
                        tableBody.appendChild(tableRow);
                      }
                    }

The code is structured into three functions.

The getValue() Function

In the HTML file the "Buzz It" button has an event listener attached. When the button is clicked the getValues() function is called. This function starts by grabbing the input from the fizz value box and storing the data in the variable fizzValue. The buzz value is also stored in the respective buzzValue variable. Next the data is converted from string to integer using the parseInt() function. An if statement runs to validate that the data in both variables are integers. If both are integers the variables are passed to the FizzBuzz function and the returned result is stored in the variable fbData, if they are not integers an alert will trigger informing the user to enter only integers.

The FizzBuzz() Function

This function accepts two parameters, the fizz and buzz values. When this function is called the first action that takes place is returnArray is created and assigned an empty array to store the fizz buzz data. Next a for loop starts at 0 and iterates all the way to 100. During each iteration an if statement determines if the current number is divisible by the fizz value, buzz value, or both. If the number is divisible by the fizz value then the string "Fizz" is pushed into the empty array. If the number is divisible by the buzz value then the string "Buzz" is pushed into the array. If the number is divisible by both then the string "FizzBuzz" is pushed into the array. Finally, if none of these conditions are met the number or index of the current iteration is pushed into the array. This creates the data we will need to display on the page. Finally, the returnArray is returned and will be used as an argument when the displayData() function is called.

The Display() Function

This function takes one argument. The fbData array is passed to the function. Next using document.getElementById and targeting the "results" id, we can grab the table element and save it to the variable tableBody. We use the same technique and target the id "fbTemplate" to get the template element and save the data to the variable templateRow. Before moving forward creating the table we clear it by setting the inner HTML of the tableBody variable to an empty string. Next we run a loop starting at 1 and iterating to the end of the fbData array incrementing by 5. This allows us to create the table data for each item building 5 columns at a time.

The way we set the data is by first importing the template content using importNode and then storing that data in the variable tableRow. Next we get the columns of the template using querySelectorAll and targeting the "td"element. This data is stored in the variable rowCols. Using the rowCols variable we add a class list with value of the current item. For example if the current item is "Fizz" then the respective class will be added. Back in the CSS document there are styles for Fizz, Buzz, and FizzBuzz to change the way the font color for each is displayed. rowCols is also used to set the text content equal to the current item data. This is done for 5 columns during each iteration, creating the full table of 100 items, 5 columns in each row.Finally the tableBody has template data for each row appended to the child element. The final result is the a table of numbers from 1-100 with Fizz, Buzz, or FizzBuzz displayed on the page!
